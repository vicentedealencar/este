This is a fork of [Este.js](https://github.com/Steida/este) intended to support desktop applications thanks to [Electron](http://electron.atom.io/).

## Get ready

`npm install -g gulp`
`npm install -g electron-prebuilt`

`npm install`

## Go!

run Este stuff

`gulp`

open your application window

`electron .`
